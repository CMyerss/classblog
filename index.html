This is where the article taught me about <br>. To quickly define the <br> tag it allows us to be able to advance to the next line simply inside our current <p> tag. with the <br> tag we can make a line continue onto the next without trying to backdoor any html code or do some weird unreadable code method! For partners, I would advise to check in with like a form or something that their partner can't see to see if everyone is pulling their weight. I know in the past this has been an issue in groups I've been in and being able to communicate that is nice. Very happy I didn't have that in this class. I think what most helped me learn the material of this class was the book and the labs. Without those I don't think I would have been able to do any of the PAs.
