Blog Post 1: HTML Basics, on Geeks for Geeks.

  • Summary

    • I first wanted to start off with a basic article about everything simple on HTML. I personally have never worked on anything HTML before, and I knew Geeks for Geeks is a great website to get started, so, here we are! The first thing that I found to be very important is the body tags. In the Labs my partner and I had been getting away without them, but now I see due to simple grouping and it's important so that when we get to complex topics, we can be aware of what we want to show.

    • One of the coolest things I learned in this article, and will be what I showcase in the code sample is a feature of the <p> tag. When writing paragraphs there may come a time where we want to use the paragraph tag but have the next line carry over. At first, I was very confused on how to do this, I tried inserting a lot of spaces, tabs, more p tags, but soon came to realize that this is simply not the way to do it. This is where the article taught me about <br>. with the <br> tag we can make a line continue onto the next without trying to backdoor any html code or do some weird unreadable code method!
