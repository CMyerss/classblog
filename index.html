    • I first wanted to start off with a basic article about everything simple on HTML. I personally have never worked on anything HTML before, and I knew Geeks for Geeks is a great website to get started, so, here we are! The first thing that I found to be very important were the body tags. In the Labs my partner and I had been getting away fine without them, but now I see due to grouping of components as well as later on it's important when we get to complex topics, so that we as programmers can be aware of what is displaying on the page and what we do not want to display on the page. The article did a great job, in all of it's code examples, with showing how to body tag should be used, and more specifically why it should be used which I really appreciated.

    • One of the coolest things I learned in this article, and will be what I showcase in the code sample is a feature of the <p> tag. When writing paragraphs there may come a time where we want to use the paragraph tag but have the next line carry over. At first, I was very confused on how to do this, I tried inserting a lot of spaces, tabs, more p tags, but soon came to realize that this is simply not the way to do it. This is where the article taught me about <br>. To quickly define the <br> tag it allows us to be able to advance to the next line simply inside our current <p> tag. with the <br> tag we can make a line continue onto the next without trying to backdoor any html code or do some weird unreadable code method! Here is an example:

  • Sample Code

    • <body>
      Top <br>
      Middle<br>
      Bottom <br>
      </body>
